Continuous Integration

This document provides an overview of the various Continuous Integration Tools

Overview

Tool Description
Codacy is an automated code analysis/quality tool that helps developers ship better software, faster.
Codebeat is an automated static code analysis tool.
Codeclimate is a command line interface for the Code Climate analysis platform.
CodeQL is the code analysis engine developed by GitHub to automate security checks.
